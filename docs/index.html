Customer Liability Coverage Reconciliation

A Python portfolio demonstration that compares a synthetic customer-liability ledger against public on-chain wallet balances, computes coverage ratios, and renders review-ready treasury reports.

View Repository Architecture Notes

Liability Aggregation

Generates a deterministic synthetic ledger and aggregates customer balances by asset with DuckDB SQL.

Reserve Checks

Fetches public BTC, ETH, USDC, and USDT balances from mempool.space and Etherscan.

Exception Reporting

Flags OK/WATCH/BREACH statuses and renders both an HTML dashboard and Excel workbook.

This is a portfolio demonstration. The customer ledger is synthetic, public wallets are demo data, and this is not an audit of any real exchange.
